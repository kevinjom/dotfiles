## Dotfiles

My dotfiles


### Usage

1. Clone this repo

```
git clone https://github.com/kevinjom/dotfiles.git
```

2. Run the scripts to apply changes to your system

```
./bin/apply_dotfiles.sh # apply the dotfiles
./bin/apply_brew.sh  # install the missing brew formulas
```
